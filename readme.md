
# Notas:

```
Para ejecutar el prrgrama doble click en consola y escribir:

node renombra.js -f= cadenaTextoBuscar -r= cadenaTextoReemplazar

Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -f, --find     Indica texto coincidente a reemplazar
                                              [cadena de caracteres] [requerido]
  -r, --replace  Indica el nuevo texto para el archivo
                                              [cadena de caracteres] [requerido]
```
